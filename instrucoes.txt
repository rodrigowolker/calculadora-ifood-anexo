
# Instruções para publicar no Streamlit Cloud

1. Crie uma conta no Streamlit Cloud: https://streamlit.io/cloud
2. Crie um repositório no GitHub (público).
3. Envie os arquivos:
   - app.py
   - requirements.txt
4. (Opcional) Adicione sua logo na mesma pasta, mas este código já está funcional sem logo.
5. No painel do Streamlit Cloud, clique em "New App".
6. Escolha o repositório, branch e o arquivo principal (app.py).
7. Clique em "Deploy" e aguarde.
8. Seu app estará online com link: https://seuapp.streamlit.app
