# Instruções para publicar no Streamlit Cloud

1. Crie uma conta no Streamlit Cloud: https://streamlit.io/cloud
2. Crie um repositório no GitHub (público).
3. Envie os arquivos:
   - app.py
   - requirements.txt
   - Ativo 1.png (sua logo)
4. No painel do Streamlit Cloud, clique em "New App".
5. Escolha o repositório, branch e o arquivo principal (app.py).
6. Clique em "Deploy" e aguarde.
7. Seu app estará online com link: https://seuapp.streamlit.app
